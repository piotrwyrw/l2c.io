To define a function in the C programming language, you must use the following syntax:
type
A primitive of complex type.
name
The function identifier. A C lexical token.
statement-list
A sequence of statements and/or expressions.
Unless a function's type is void, it must contain a return statement that will terminate the execution of the function and will result in the function call expression being evaluated to the provided expression.
