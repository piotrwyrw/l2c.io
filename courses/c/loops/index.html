Loops

The C programming language offers two main types of loops, which are the for and while loops.
The behaviour and syntax of these loops is (in most cases) directly compatible with other languages of the C family, such as Java or C++.
As a refresher, the while loop will keep repeating the enclosed code block for as long as the condition evaluates to true. The for loop has a slightly more complicated syntax. Even if the loop is seemingly endless according to the condition(s) in the statement, you may still forcibly break out of that loop at any point using the break statement. You may also skip one iteration of the loop at will, using continue.
