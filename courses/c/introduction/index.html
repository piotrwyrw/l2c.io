ANSI C Programming Course

Welcome to the ANSI C programming course!

The [ANSI] C programming language had been used to create pieces of software like the Linux, Mac and Windows operating system kernels, compilers and interpreters for languages like python, databases (eg. MySQL) and many others. Although many new languages, such as Python, are making their ways into all kinds of programming projects, C is still widely used for programming low-level, high-performance systems, such as bootloaders or operating system(s) [kernels].
