Memory

In C, by default, all variables you declare, are stored on a small block of memory, called the stack.
The stack is also the place where the return addresses of the functions are stored. You can query the memory address of a variable using the & operator as follows:
Knowning the address, you could then modify the value of a variable.
You could also assign an integer value to a pointer to modify a known part of memory, which is very useful in low-level and kernel development.
In case you need to use a large block of memory to persistently store data, you will need to use the malloc function from the standard library to dynamically allocate a part of memory. To prevent your programs from using up all of your system memory, remember to free the allocated memory block, as soon as you don't need them anymore.
